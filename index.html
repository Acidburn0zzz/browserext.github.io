--- title: Browser Extension Community Group --- {% include warning.html %}

This W3C Community Group provides a lightweight venue to facilitate discussion between Web Browser vendors, as well as other interested parties, in order to establish a set of standards for interoperable browser extensions.

Several browser vendors have already started converging towards a common approach. This CG aims to ensure actual interoperability rather than mere similarity.
